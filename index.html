stat-nlp-book

About

stat-nlp-book project was an interactive Statistical Natural Language Processing book created for the purpose of the COMPGI19 course at University College London. The interactivity of the book is achieved via Moro and Wolfe. The current project pages are static, dumped HTML code. If you're interested in building your own interactive book, please follow the instructions here.

The project was actively pursued in 2015/16, but is as of early 2016 deprecated in the current form. Stay tuned for new version.

Find the source code here.

The project has been co-sponsored by an UCL Arena small grant.
